System Flow

This system allows residents and customers to report electricity-related issues such as power outages, electrical faults, damaged power lines, transformer issues, and unstable voltage through the official Facebook Page Messenger.

  1. Messenger Trigger: The user sends a message to the official Facebook page.
  2. Auto Reply with Form Link: The system automatically replies with a link to the reporting form (Open report form).
  3. Form Submission: The user fills in their name, contact details, issue description, and exact location.
  4. GPS Capture: The system captures latitude and longitude for accurate location tracking.
  5. Data Storage: The report is saved automatically to the Supabase database.
  6. Dashboard Mapping: The admin dashboard retrieves submitted reports and shows each one as a marker on an interactive map for faster field response.
